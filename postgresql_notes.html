1. install postgres
apt install postgresql postgresql-contrib

2. check if postgresql daemon is active
service postgresql status

3. use postgresql cli
sudo -u postgres psql

4. check postgresql connection info
\l

5. list of all users with privileges
\du

6. set password to default postgres user. For local testing, setting pw to password
\password password

7. create a test database
CREATE DATABASE test_db;

8. connect to the test_db database
\c test_db

9. quit psql
\q

10. Explore postgres server with pg admin

Under general:
name = localhost

Under connection:
host = localhost
port = 5432
maintenance database = postgres
username = postgres
password = password

11. For future turning on and off the server
check status postgres
sudo systemctl status postgresql

start postgres server
sudo systemctl start postgresql

stop postgres server
sudo systemctl stop postgresql
