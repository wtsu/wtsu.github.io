1. install postgres
apt install postgresql postgresql-contrib

2. check if postgresql daemon is active
service postgresql status

3. use postgresql cli
sudo -u postgres psql

4. check postgresql connection info
\l

5. list of all users with privileges
\du

6. set password to default postgres user. For local testing, setting pw to password
\password postgres

7. create a test database
CREATE DATABASE test_db;

8. connect to the test_db database
\c test_db

9. quit psql
\q

10. Explore postgres server with pg admin

1. Install the public key for the repository (if not done previously):
curl -fsS https://www.pgadmin.org/static/packages_pgadmin_org.pub | sudo gpg --dearmor -o /usr/share/keyrings/packages-pgadmin-org.gpg
2.Create the repository configuration file: sudo sh -c 'echo "deb [signed-by=/usr/share/keyrings/packages-pgadmin-org.gpg] https://ftp.postgresql.org/pub/pgadmin/pgadmin4/apt/$(lsb_release -cs) pgadmin4 main" > /etc/apt/sources.list.d/pgadmin4.list && apt update'

3. Install for both desktop and web modes:
sudo apt install pgadmin4

Under general:
name = localhost

Under connection:
host = localhost
port = 5432
maintenance database = postgres
username = postgres
password = password

11. For future turning on and off the server
check status postgres
sudo systemctl status postgresql

start postgres server
sudo systemctl start postgresql

stop postgres server
sudo systemctl stop postgresql
